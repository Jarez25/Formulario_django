Comandos de Django

Comando Descripción
python manage.py startproject <nombre_del_proyecto> Crea un nuevo proyecto de Django.
python manage.py startapp <nombre_de_la_aplicación> Crea una nueva aplicación dentro del proyecto.
python manage.py runserver Inicia el servidor de desarrollo para ejecutar la aplicación localmente.
python manage.py makemigrations Crea archivos de migración para los cambios en los modelos.
python manage.py migrate Aplica las migraciones pendientes a la base de datos.
python manage.py createsuperuser Crea un superusuario para acceder al panel de administración.
python manage.py shell Inicia una shell interactiva de Python con acceso al proyecto.
python manage.py test Ejecuta las pruebas unitarias de la aplicación.
python manage.py collectstatic Recopila archivos estáticos en una ubicación central.
python manage.py flush Elimina todos los datos de la base de datos.
python manage.py dumpdata Exporta datos de la base de datos en formato JSON o YAML.
python manage.py loaddata Carga datos en la base de datos desde un archivo.
python manage.py showmigrations Lista todas las migraciones y su estado.
python manage.py check Verifica el proyecto en busca de problemas de configuración y código.
python manage.py changepassword <nombre_de_usuario> Cambia la contraseña de un usuario.
python manage.py createsuperuser Crea un superusuario interactivo.
python manage.py shell_plus Inicia una shell interactiva con funcionalidad adicional proporcionada por la extensión "django-extensions."
python manage.py runserver <puerto> Inicia el servidor de desarrollo en un puerto específico.
python manage.py dbshell Accede a la línea de comandos de la base de datos.
